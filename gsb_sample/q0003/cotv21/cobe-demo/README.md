# cobe-demo

基于 Vite + React + TypeScript 和 cobe（WebGL）的交互式地球 demo

## 运行环境
- Node.js (推荐 v18 或以上)
- npm 或 yarn 包管理器

## 依赖安装
```bash
npm install
```

## 开发命令
```bash
npm run dev
```
开发服务器将在 http://localhost:5173 启动

## 生产构建命令
```bash
npm run build
```
构建产物将输出到 `dist` 目录

## 实现功能说明

### 任务 A - 旋转与拖拽
- **自动旋转**：无操作且无惯性时，地球持续低速绕竖轴旋转
- **水平拖拽**：鼠标/触摸水平拖拽改变绕竖轴朝向，符合地图拖拽直觉
- **垂直拖拽**：鼠标/触摸垂直拖拽改变俯仰，限制在合理区间（-0.8 到 0.8 弧度）
- **指针捕获**：拖拽过程中指针移出画布仍能完成操作
- **惯性效果**：拖拽停止后保持惯性并逐渐停稳
- **自动旋转恢复**：停稳后恢复拖拽前的自动旋转

### 任务 B - 缩放功能
- **滚轮缩放**：
  - 滚轮向上（滚轮方向向上）：放大地球
  - 滚轮向下（滚轮方向向下）：缩小地球
  - 缩放范围：0.5x - 2x
- **双指缩放**：
  - 双指捏合：缩小地球
  - 双指张开：放大地球
  - 缩放范围：0.5x - 2x
- **触摸操作**：
  - 单指：控制旋转（水平+垂直）
  - 双指：控制缩放
  - 操作职责清晰划分

### 任务 C - 球面元素动画
- **动态标记**：上海位置的标记大小呈现周期性变化（正弦动画）
- **静态对照**：旧金山位置的标记保持大小不变，作为对照
- **地理含义**：城市标记位置保持不变，符合基线地理含义

## 验证方法

### 1. 准备环境
```bash
git clone <repository-url>
cd cobe-demo
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 在浏览器中验证
打开浏览器访问 http://localhost:5173

#### 验证任务 A
- 观察地球是否持续自动旋转
- 尝试水平拖拽：地球应跟随手势旋转
- 尝试垂直拖拽：地球应改变俯仰角度
- 观察拖拽停止后的惯性效果
- 等待惯性停稳后，观察是否恢复自动旋转

#### 验证任务 B
- 使用鼠标滚轮：向上滚动放大，向下滚动缩小
- 在触摸设备上使用双指手势：捏合缩小，张开放大
- 验证缩放范围限制在 0.5x - 2x 之间

#### 验证任务 C
- 观察上海位置的标记：应呈现周期性大小变化
- 观察旧金山位置的标记：应保持大小不变

### 4. 生产构建验证
```bash
npm run build
npm run preview
```
访问预览地址，验证所有功能在生产构建中正常工作
