# cobe 地球交互 Demo

这是一个基于 cobe (WebGL) 的地球可视化 Demo，实现了丰富的交互功能。

## 运行环境

- Node.js 16+
- npm 或 yarn
- 现代浏览器（支持 WebGL 和 Pointer Events）

## 依赖安装

```bash
cd cobe-demo
npm install
```

## 开发与生产构建命令

### 开发模式（启动本地服务器）
```bash
npm run dev
```

### 生产构建
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 功能说明

### 任务A：拖拽交互
- **自动旋转**：无操作且无惯性时，地球持续低速绕竖轴旋转
- **水平拖拽**：按住鼠标左键水平拖拽可改变地球朝向，符合地图拖拽直觉
- **垂直拖拽**：垂直拖拽可改变俯仰角度，限制在合理区间内，避免极点失控
- **指针捕获**：指针移出画布后仍可完成拖拽
- **惯性效果**：抬起鼠标后有惯性并逐渐停稳，停稳后恢复自动旋转
- **无叠加**：拖拽过程中不叠加自动旋转与惯性

### 任务B：缩放功能
- **滚轮缩放**：
  - 滚轮向上：缩小地球
  - 滚轮向下：放大地球
  - 阻止默认页面滚动行为
- **双指缩放**（触摸设备）：
  - 双指捏合（间距缩小）：缩小地球
  - 双指张开（间距增大）：放大地球
  - 缩放范围：0.5x - 2.5x
- **单指与双指职责**：
  - 单指：控制地球旋转和俯仰
  - 双指：控制地球缩放

### 任务C：周期性球面元素变化
- 上海标记（[31.2304, 121.4737]）的大小会周期性变化
- 旧金山标记（[37.7595, -122.4367]）保持稳定作为对照
- 地理含义保持与基线一致

## 验证方法

### 步骤1：拉取代码并安装依赖
```bash
git clone <仓库地址>
cd cobe-demo
npm install
```

### 步骤2：启动开发服务器
```bash
npm run dev
```
在浏览器中打开显示的本地地址（通常是 http://localhost:5173）

### 步骤3：验证任务A
1. 观察地球是否自动绕竖轴旋转
2. 按住鼠标左键水平拖拽，验证地球朝向是否跟随鼠标移动
3. 垂直拖拽鼠标，验证俯仰角度是否变化且不会失控
4. 拖拽时快速移出画布再松开，验证拖拽是否完成
5. 快速拖拽后松开，验证惯性效果
6. 等待惯性消失，验证是否恢复自动旋转

### 步骤4：验证任务B
1. 将鼠标悬停在地球上，滚动滚轮：
   - 向下滚动：地球应放大
   - 向上滚动：地球应缩小
   - 验证页面不随滚轮滚动
2. （触摸设备）使用双指：
   - 双指张开：地球应放大
   - 双指捏合：地球应缩小
3. 验证缩放范围在 0.5x 到 2.5x 之间
4. 验证单指和双指操作互不干扰

### 步骤5：验证任务C
1. 观察上海标记（亚洲位置）的大小，验证其周期性变化
2. 观察旧金山标记（北美洲位置），验证其大小保持稳定
3. 确认两个标记的位置正确

### 步骤6：验证生产构建
```bash
npm run build
```
确认构建成功，无错误。
