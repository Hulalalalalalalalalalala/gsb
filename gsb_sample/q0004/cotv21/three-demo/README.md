# 3D Scroll Demo

一个使用 Three.js 打造的沉浸式三维滚动体验网页。

## 环境要求

- Node.js 16+ 
- npm 或 yarn 包管理器
- 现代浏览器（支持 WebGL）

## 安装与运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

运行后访问 `http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 功能说明

### 三维场景特性
- **主物体**: 细分二十面体，使用物理材质添加质感和清漆效果
- **线框附属体**: 半透明线框环绕主物体，随滚动进度变化透明度
- **粒子系统**: 1000+ 粒子构成的星云效果，随滚动旋转和淡出
- **灯光系统**: 四盏动态灯光（主光源、补光、轮廓光、环境光）
- **雾化效果**: 场景深度雾，增强层次感

### 滚动交互
页面的滚动进度（0-1）会驱动以下三维元素的变化：

1. **主物体**:
   - Y轴旋转: 0 → 360°
   - X轴旋转: 0 → 90°  
   - Y轴位移: 正弦波动 ±0.5

2. **线框**:
   - 反向旋转，创造层次感
   - 透明度: 0.3 → 0.7

3. **粒子**:
   - 缓慢旋转
   - 透明度: 0.6 → 0.2（逐渐淡出）

4. **相机**:
   - 从正前方逐渐移动到侧面视角
   - 相机距离: 6 → 4（逐渐拉近）

5. **灯光**:
   - 主光源强度减弱
   - 补光和轮廓光强度增强
   - 创造动态光影变化

6. **雾效**:
   - 浓度逐渐增加，营造深度感

### 页面结构
- **首屏**: 大标题、副标题、滚动提示
- **中间两屏**: 玻璃拟态卡片，包含特征介绍和统计数据
- **末屏**: 项目信息、运行命令、技术栈说明

### 响应式设计
- 小屏幕（<768px）: 卡片居中，网格布局自动折行
- 超小屏幕（<480px）: 单列布局，内容居中

## 逐步验证方式

1. 打开页面后，观察初始三维场景
2. 向下滚动鼠标滚轮或拖动滚动条
3. 注意观察：
   - 主物体的旋转和位置变化
   - 线框的透明度变化
   - 粒子系统的旋转和淡出
   - 相机视角的移动
   - 整体光影氛围的变化
4. 滚动到底部时，观察三维场景的最终状态

## 技术栈

- **构建工具**: Vite

- **类型系统**: TypeScript
- **三维引擎**: Three.js

## 关键技术点

1. **React + Three.js 集成**: 使用 useRef 和 useEffect 管理 Three.js 生命周期
2. **滚动监听**: 使用 window.scroll 事件获取滚动进度
3. **平滑插值**: 滚动进度使用缓动算法，避免抖动
4. **资源清理**: 组件卸载时正确清理 Three.js 资源，防止内存泄漏
5. **性能优化**: 使用 ResizeObserver 监听窗口尺寸变化
6. **玻璃拟态**: 使用 backdrop-filter 和半透明背景实现毛玻璃效果

## 注意事项

- 确保浏览器支持 WebGL
- 建议使用现代浏览器（Chrome, Firefox, Safari）获得最佳体验
- 滚动时可能会有轻微的性能影响，属于正常现象
- 移动端体验可能会受到设备性能限制
